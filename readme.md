## vehicle_management

## 无人停车

该项是基于QT 5.11.1(Qt Creator 4.6.2)平台开发的《无人停车场》仿真系统

**PS：**
1. 百度AIAPI接口
1.1 你需要先申请一个用于识别图片的百度AI接口
1.2 build项目的.pro文件
1.3 点击Sources/widget.cpp文件
1.4 在230行下  QString中补全百度AI接口
（我使用得是tokenAPI接口）

2. 需要自己新建一个数据库（数据库的名字  可以自己定义，定义后 需要修改mydb.cpp中的对应名称）
